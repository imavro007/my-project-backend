
# হবিগঞ্জ জেলা রোভার পোর্টাল - প্রোডাকশন গাইড

এই প্রকল্পটি এখন একটি পূর্ণাঙ্গ ফুল-স্ট্যাক (Full-stack) অ্যাপ্লিকেশন। এটিকে গ্লোবাল করার জন্য নিচের ধাপগুলো অনুসরণ করুন:

## ১. লোকাল কম্পিউটারে চালানো
১. `node server.js` কমান্ড দিয়ে ব্যাকএন্ড সার্ভার চালু করুন (প্রয়োজনীয় প্যাকেজ: express, mongoose, cors, jsonwebtoken, bcryptjs, dotenv)।
২. আপনার ফ্রন্টএন্ড থেকে ডেটা এখন সরাসরি সার্ভারের মাধ্যমে আসবে।

## ২. গ্লোবাল ডাটাবেস (MongoDB Atlas)
১. [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) এ একটি ফ্রি ক্লাস্টার তৈরি করুন।
২. আপনার কানেকশন স্ট্রিংটি সংগ্রহ করুন।
৩. আপনার সার্ভারে বা `.env` ফাইলে `MONGODB_URI` হিসেবে এটি সেট করুন।

## ৩. ডেপ্লয়মেন্ট (Deployment)
- **Frontend:** Vercel বা Netlify-তে আপনার রিয়েক্ট কোড আপলোড করুন। 
- **Backend:** Render.com বা Railway.app এ `server.js` ফাইলটি ডেপ্লয় করুন।
- **URL Update:** ডেপ্লয় করার পর `services/db.ts` এবং `pages/Login.tsx` ফাইলে `http://localhost:5000` এর বদলে আপনার ডেপ্লয় করা সার্ভারের ইউআরএল বসিয়ে দিন।
